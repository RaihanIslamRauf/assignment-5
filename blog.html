Important FAQ

1. What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a blueprint that allows developer's to access and modify the elements of a web page.

2. How do you select an element from the DOM?

There are several methods. For example: getElementById() is used for selecting unique ID's, getElementsByClassName() or getElementsByTagName() is used for multiple elements and querySelector() and querySelectorAll() is used for CSS-style selections.

3. What is event delegation in the context of the DOM, and why is it useful?

In the context of DOM, an event delegation is a technique where a single event listener is added to a parent element, allowing it to handle events from child elements that bubble up. It is useful because, it reduces memory usage, it automatically applicable to new child elements and it centralizes event handling for easier management.

4. How do you manipulate an element's attributes and styles using the DOM?

Using the DOM, we can manipulate an element's attributes and styles using JavaScript. For example: Attributes: element.getAttribute('attrName'), element.setAttribute('attrName', 'value') and Styles: element.style.property = 'value', element.classList.add('className'), element.classList.remove('className').
